Inline-flex container of Avatar primitives. Order in DOM = visual right-to-left; left-most renders on top via flex-direction: row-reverse.
ring
2px solid in the surface color (white on canvas, Grey-100 on sunken, Blue-900 on navbar). Acts as a "cutout" so adjacent avatars read as distinct shapes.
overlap
-4 / -6 / -8px for xs / sm / md.
overflow
Show ~5 avatars then a "+N" chip — same shape, Grey-200 background, indigo text. Use role="button" + aria-label so screen readers announce it as expandable. Clicking opens a dropdown listing all members.
a11y
Each child Avatar carries aria-label="Full name". The group itself doesn't need a label — children carry context.
