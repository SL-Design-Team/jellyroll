row anatomy
Vendor icon · Name (ExtraBold) · meta line (type · snaplex · creator) · health badge · chevron. Health pill: Connected green · Auth failed red · Untested grey.
grouping
Connections grouped by snap/vendor type. Most-recent and most-used surface first within each group.
create-new
Sticky footer row "+ Create new connection…" — opens a Drawer with the connection-creation form for that snap type. Always available, never hidden.
health
Background-tested every N minutes; badge reflects last result. Click a failed connection to re-test or open its config.
scope
Picker filters to connections that are valid for the current snap type by default. "Show all" toggle in the search header to override.
a11y
role="combobox" aria-haspopup="listbox". Menu rows: role="option". Vendor icons are decorative (aria-hidden).
