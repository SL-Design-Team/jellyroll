Use a sequential ramp when one variable goes from "low" to "high". Pair the axis with neutral grey ticks.
When to use which ramp
  • Pink — engagement, activity, growth metrics
  • Cyan — throughput, request count, default magnitude
  • Mint — duration, latency, performance metrics
