contract
Every input primitive must visually implement at least: default, hover, focus, disabled, read-only, error. Should implement: warning, success, loading. May implement: AI generating, AI review (Wave 3).
tokens
Border colors via --field-border-*. Background via --field-bg-*. Text via --field-text-*. Focus halo via --ring-focus. Never invent values — extend the token set if a new state appears.
validation timing
Inline async validation: debounce 350ms (--validate-debounce) before showing loading state. On-blur: validate immediately. On-submit: validate the whole form, focus the first invalid field, announce summary in a live region.
error precedence
When a field could match multiple states, precedence is: Error → Warning → AI review → Loading → Success → Focus → Hover → Default.
