structure
Header row · N data rows · footer with Add button(s). Each data row has a left grab handle and a right trash icon.
add
Primary add button is always visible. Multiple add variants are allowed: Add header · Add from clipboard (parses pasted key:value pairs) · Auto-map by name · SnapGPT map.
reorder
Drag handle (grip-vertical) on the left. Dragging row gets opacity: 0.7 + Blue-100 background. Drop indicator (1px Blue-600 line) between rows.
remove
Trash icon on hover. Confirm only for rows that have non-empty input or are referenced downstream — empty rows delete silently.
empty state
Single message + inline "Add one" link. Don't show the header row when empty.
counter
Right-aligned in the label: 3 of unlimited · 4 of 32 max. Turns red over the cap.
a11y
Each row should be focusable as a unit. Handle: role="button" aria-label="Drag to reorder". Use aria-live="polite" when announcing add/remove.
