placement
Context decides. At the page level in the page header when it scopes the whole view (dashboard, all charts at once); in the upper right-hand corner outside a table (aligned with its title or above the header row); or inside the table actions row beside search and filters when it scopes just that table.
presets
Quick windows live as segments. Configure the set per context — 24h / 7d / 30d / 12mo is the dashboard default. As few as two segments (down to one preset + More) up to five.
more
Trailing segment with a hairline divider and caret. Opens the Menu with period-relative ranges (Last month, Last quarter) and Custom range…
custom range
Reuses the Date picker two-month range calendar — round endpoints, Blue-100 tint between, range readout, Cancel / Apply. No preset sidebar; presets live in the More menu.
active value
One range active at a time. A non-preset choice moves selection onto More, which shows the value (12 May – 19 May) with its caret; presets deselect.
sizes
Small (24px) is the default. Default (30px) for roomier page headers.
a11y
Control is a role="group" of toggle buttons; the active segment carries aria-pressed. More carries aria-haspopup="menu" + aria-expanded. Calendar follows the Date picker dialog/grid semantics.
