tokens
Selected values render inside the control as removable chip-tokens (Blue-100 / Blue-700, X to remove). Tokens wrap to multiple lines as needed — the control grows in height.
menu
Each row is a checkbox. Multiple selections allowed. Footer action bar: "Select all" + "Clear" (or "Select N matching" when filtered).
search
Typing in the input filters the menu in-place. Matching substrings bolded. Backspace removes the last token when the input is empty.
keyboard
↑/↓ navigate options · Space toggle · Enter close · Esc close without changes · Backspace in empty input removes last token.
a11y
role="combobox" aria-multiselectable="true". Menu: role="listbox" aria-multiselectable="true". Each option: role="option" aria-selected. Token X: aria-label="Remove {value}".
