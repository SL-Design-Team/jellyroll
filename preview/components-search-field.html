States
⌘ K
Sizes
icons
Leading magnifier 16px, subtle grey. Replaced with spinner during async debounced search (350ms). Trailing X appears only when input has content.
debounce
Default 350ms (--validate-debounce) before firing the search request. Adjust per use case — instant for local lists, longer for remote APIs.
type
Use type="search" — provides native clear button on some platforms (which is hidden by the system's CSS reset; the explicit clear button replaces it).
shortcut
Global search fields show ⌘K / Ctrl+K badge inline. Pressing the shortcut opens the Search overlay component.
a11y
Wrap in <form role="search"> if the field is the primary search affordance on the page. Clear button: aria-label="Clear search".
