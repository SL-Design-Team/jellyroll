States
Strong password
Weak — add length and variety
toggle
Trailing eye button switches type between password and text. aria-label reflects current action ("Show password" / "Hide password"). Default state: hidden.
strength
4-bar meter, color-coded: weak (red) · fair (orange) · good (yellow) · strong (green). Use only on set-password flows, not sign-in.
autocomplete
Always set autocomplete="current-password" for sign-in, "new-password" for set/change. Lets password managers help.
paste
Never block paste — it's an accessibility anti-pattern (breaks password managers).
