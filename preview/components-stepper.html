Stepperprominent −/+ number input · for small ranges
States
Minimum 0.
Maximum 10.
Must be 0 or greater.
Sizes
use
Small integer ranges (retries 0–10 · concurrency 1–8 · seats). For free-form numbers (timeout in ms, currency), use Number input.
chrome
Three segments: − button · centered input · + button. 1px divider between segments. Buttons disable at min/max.
behavior
Click increments by step (default 1). Hold for repeat with acceleration. Direct typing allowed; on blur, clamp to min/max and show error if invalid.
a11y
Input has type="number" role="spinbutton" aria-valuemin aria-valuemax aria-valuenow. Buttons: aria-label="Increment / Decrement". ↑/↓ keys also adjust.
