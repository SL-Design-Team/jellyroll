variants
Neutral default · Info noteworthy / active · Success positive · Warning sub-optimal · Danger critical · Tier product tier
structure
Container + optional leading icon (left-aligned) + optional text. Avoid long strings — truncate with tooltip if needed.
behavior
Static by default. Use <a class="sl-badge"> to make actionable — hover underlines and dims slightly.
a11y
Icon-only: role="img" + aria-label="…" describing the metadata, not the icon. Icon-with-text: aria-hidden="true" on the icon. Inline meta-info: add sr-only text if the badge's meaning isn't obvious from context.
use
System-generated metadata (status, tier, count, type). If user-created or customizable → use a Tag instead. If standalone/floating with no object → use plain text or an icon.
