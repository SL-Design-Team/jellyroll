seconds
Must be between 1 and 3600.
type
type="number" · also set inputmode="numeric" for mobile keyboards. Use step, min, max for native clamping.
steppers
24px wide column on the right. Hover-elevate, click increments by step (default 1). Hold for repeat. Hidden if readonly.
alignment
Numbers right-aligned by default (units / currency reads cleanly). With prefix, switch to left-aligned (.sl-number--left).
monospace
Input text uses var(--font-mono) so digits line up in tables and series.
clamping
On blur, snap to min/max and show inline error if invalid. Don't silently rewrite the user's typed value while they're typing.
