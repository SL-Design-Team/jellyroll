variants
Inline for small, single-file uploads (avatar, signature). Drop zone for primary upload moments (pipeline import, schema load, asset upload).
accept
Hint string ("JSON, YAML, or .slp · up to 50 MB") is always visible. Mirror it as accept attribute on the input + enforced client-side validation.
drag states
Default border Grey-400 dashed. On dragover: border + background shift to Blue-100. Title updates to "Release to upload N files".
file list
Each file is a row with icon · name · meta · cancel/remove. States: uploading (blue border + progress bar), success (green border + check), error (red border + tint + reason).
progress
4px bar inside the file row. Shows percent + size pair ("8.1 MB of 24.3 MB · 33%"). Cancel button is the same X.
a11y
Drop zone has a focusable button inside ("browse"). The dropzone itself is role="region" aria-label="Upload files". Announce progress via aria-live="polite".
