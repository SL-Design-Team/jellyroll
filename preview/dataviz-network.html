States selected · hover-focus · zoomed-out overview
Selected
Dashed halo around the node; 1-hop edges + neighbours stay full opacity, everything else dims to 0.25.
Hover
Brighter ring on the hovered node; tooltip shows name + count. No edges dimmed — just a hover hint, not a focus state.
when
Visualizing many-to-many relationships where structure matters more than scale: pipeline dependency graphs, snap-to-account relationships, asset lineage. Not for hierarchies (use a tree) or linear flows (use a Sankey).
node
Filled circle. Size scales with degree (incoming + outgoing edges), capped at min 8px / max 26px so even ones don't disappear. Color indicates cluster, status, or category — never both at once. Labels appear on hover, on the largest 3–5 hub nodes, or on selected nodes.
edge
2px line, stroke-linecap round, colored to match the source cluster at 0.55 opacity. Cross-cluster edges use the source cluster's color slightly darker. No arrowheads in undirected graphs; arrowheads in directional ones (lineage).
clusters
Use up to 5 categorical colors from the dataviz palette (Orange / Yellow / Green / Blue / Purple). Avoid red — reserve for failed/error nodes. Background is the standard dot-grid white canvas.
selection
Dashed halo around the selected node + its 1-hop neighbours stay full opacity; everything else dims to 0.22 opacity. Click empty canvas to clear. Persists through hover.
hover
Solid ring brightens around the hovered node. Tooltip shows name + degree + cluster. No global dimming — hover is a hint, not a focus mode.
interactions
Drag a node to reposition (force layout re-stabilizes). Scroll to zoom (10%–400%). Right-click on a node for context actions (open, expand neighbours, exclude). Double-click background to reset zoom.
performance
Force layout (d3-force or equivalent) up to ~500 nodes / 2000 edges. Beyond that, group into super-nodes by cluster and expand on click. Never render labels for all nodes simultaneously.
variants
For axial / categorical graphs where structure repeats per category, use the Hive plot variant (separate card). For directional time-ordered flows, use Sankey.
a11y
Provide a parallel table view (toggle in toolbar) listing nodes + neighbours — the graph alone isn't screen-reader friendly. Each node has role="button" aria-label="{name}, degree {n}, cluster {x}". Color encoding is supplemented with shape (square = pipeline, circle = snap, diamond = account) when used as the only differentiator.
