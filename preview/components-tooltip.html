Tooltip vs alt text
Situation Use Why
Icon-only button Tooltip + aria-label Label is available to screen readers via aria-label; tooltip serves sighted mouse users.
Decorative image alt="" Empty alt hides it from assistive tech. A tooltip on a purely decorative element adds noise.
Informative image alt="…" Alt text is always available — tooltips require hover and are invisible on touch devices.
Disabled control with reason Tooltip Explains why the action is unavailable. Wrap in a focusable container so keyboard users can reach it.
Complex chart or diagram alt="…" or aria-describedby Tooltips truncate at 240px and are not discoverable by touch. Use visible caption or linked description instead.
Truncated text in a table cell Tooltip Surfaces the full string on hover. Keep content identical to the truncated text — no extra information.
