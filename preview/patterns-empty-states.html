anatomy
Round 64×64 icon medallion, 16px ExtraBold title, 13px regular description (≤2 lines, ≤140 chars), 1–2 action buttons. Centered, max-width 360px. Lives inside the container that would normally show data.
variants
First-run (brand-tinted medallion, primary CTA) · No results (grey medallion, secondary "Clear filters") · No permission (yellow medallion, ghost link) · Inline (one-line, no medallion — for sub-sections inside a populated screen).
copy
Title is a declarative phrase, sentence case, names the missing object ("No pipelines in this project yet"). Description names why + the next step. Never apologize ("Sorry, no results"). Never use "Oops".
actions
Primary CTA only when the user can resolve the empty state themselves. For no-permission, replace CTA with a ghost "Request access" link — never a disabled button. For no-results, the action is always "Clear filters" or "Adjust search", never "Create".
art
Single Lucide glyph in a flat medallion — no illustrations, no characters, no spot art. Brand-gradient tint reserved for first-run states only.
vs. loading
Show a skeleton while data is fetching. Only render the empty state once the request resolves with zero rows — never flash empty before loading completes.
a11y
Empty container has role="status" with the title as accessible name. Actions are real buttons, focusable, keyboard-reachable.
