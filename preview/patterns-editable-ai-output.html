chrome
Purple bordered card with header band. Header carries label · confidence pill · actions. Always make it visually clear this is AI-generated, not user-typed.
confidence pill
Three levels: High (green dot · >0.85) · Medium (yellow · 0.6–0.85) · Low (orange · <0.6). Encourage human review more aggressively when low.
diff view
Word-level diff between previous value and AI proposal. Adds in green (Green-100/800), deletes in red strikethrough (Red-100/700), unchanged in body color. Best for "rewrite this" interactions.
candidates
For ambiguous fields (names, labels), surface N suggestions ranked by confidence. Single radio-style select. "Use selected" applies the active one.
edit
Body is contenteditable — user can refine the proposal before accepting. Edits don't change the "AI-proposed" provenance flag.
actions
Accept commits the value, removes the AI chrome · Regenerate runs again · Discard reverts to the prior value.
