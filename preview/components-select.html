trigger
Looks like a text field with a trailing chevron-down. Rotates 180° when open.
menu
Positioned 4px below the trigger, same width. Max-height 260px (≈6 items) with scroll. Supports group labels (10px UPPERCASE mono) and dividers.
selected
Highlighted with Blue-100 background, Blue-700 text, ExtraBold. Right-side check icon. Single selection only — for multi, see Multi-select.
create
"Create new …" inline option at the bottom of the menu when applicable (blue text + plus icon). Use for connection pickers, project pickers, etc.
keyboard
↑/↓ navigate · Enter select · Esc close · Home/End first/last · type-ahead jumps to first match.
a11y
role="combobox" aria-expanded aria-haspopup="listbox" aria-controls="menu-id". Menu: role="listbox"; items: role="option" aria-selected.
virtualize
For lists with >200 options (Snaplex picker, account list), virtualize rows. Don't paint 5,000 DOM nodes.
