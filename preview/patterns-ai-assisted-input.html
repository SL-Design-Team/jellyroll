states
Idle (empty) · Filled (manually written — "Improve" replaces "Generate") · Generating (purple sweep + spinner + Cancel) · Review (purple tint + Accept/Revise buttons) · Accepted (returns to Filled, history preserved).
placement
Generate button sits trailing inside the field. For multi-line fields (textarea), button moves to top-right of the field. Always purple text on transparent background.
prompt popover
Optional intent-gathering step before generation. Shows when user wants to give a custom prompt. Skip by default — most fields have enough context.
review
Generated values always require explicit Accept. Never auto-write final values without confirmation. Revise reruns generation with the same context.
tokens
Uses --ai-tint (Blue-100 bg), --ai-accent (Blue-500 text/border). Matches Field states "AI generating" / "AI review" rows.
a11y
Generate button has aria-label="Generate with SnapGPT". Field announces state changes via aria-live="polite" ("Generating", "Review the generated value").
