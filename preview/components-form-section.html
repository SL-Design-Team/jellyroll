layouts
Split (default for settings pages) — 240px heading column + flexible fields column, 48px gap. Stacked (default for drawers and wizards) — heading-top, fields-below, narrower viewports.
head
Title 18/26 ExtraBold. Description 13/20 subtle grey, max 320px wide. Description is optional but recommended for non-obvious sections.
spacing
28px padding above/below each section. Border between adjacent sections. Fields inside spaced 20px apart.
collapsible
Optional toggle in the title area. Collapsed state hides the body, rotates the chevron −90°. Use for "Advanced" / "Notifications" / progressively-disclosed groups.
composition
Body composes Form rows and individual fields. Section doesn't own field chrome.
a11y
Use <fieldset> + <legend> when the section groups truly related controls (radios, checkboxes). For prose sections with mixed inputs, an <h3> heading and aria-labelledby on the section is enough.
