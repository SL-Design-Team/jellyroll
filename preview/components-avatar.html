content
Always the first initial of the person's display name. Single letter, uppercase, ExtraBold (800).
sizes
xs 16 · sm 24 · md 32 default. Never larger — we use initials, not photos, so smaller marks read better.
color
Pick one of 8 (blue, teal, indigo, orange, purple, green, pink, red) by hashing the user id or name. Yellow and grey are excluded — poor white-on-color contrast.
a11y
Always set aria-label="Full name" on the avatar element — the visible initial alone isn't enough for screen readers. When paired with a visible name, use aria-hidden="true".
related
Avatar group — overlapping stack for showing multiple users in tight contexts.
