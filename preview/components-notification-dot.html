structure
Absolutely positioned child of a position: relative host. Plain: 8px circle. With count: pill that grows with content.
position
Top-right by default (icons, buttons, nav items). Bottom-right for presence on avatars. Always offset slightly outside the host bounds.
ring
2px ring in the surface color via box-shadow. Visually "cuts out" the dot from the host edge.
variants
Danger (red) — needs attention, default · Info (blue) — new, neutral-positive · Success (green) — online/active · Warning (yellow) — away/idle · Neutral (grey) — offline/muted
a11y
The dot itself is pointer-events: none and decorative. The host element carries the aria-label describing the state (e.g. aria-label="Notifications — 3 unread"). Don't use a dot as the only indication of state for screen readers.
vs. Badge
Use a Notification dot when the indicator sits on top of another element. Use a Badge when the indicator is inline metadata next to text.
