key token
22px high, mono font ExtraBold 11px, white background, 1px grey border with a 2px bottom border (gives the "physical key" feel), 4px radius. Joined with + separators in mono grey. Symbols: ⌘ Cmd · ⌃ Ctrl · ⌥ Opt · ⇧ Shift · ↵ Enter · ⌫ Delete.
cheatsheet
Opens with ? from anywhere. Modal overlay (Overlay surface), 720px wide, columns grouped by scope: Global · current-product context · Tables · Editing. Filter input at top searches both labels and key combos.
scope
Cheatsheet contents change per product — Designer shows canvas shortcuts; Monitor shows table/timeline shortcuts. Global section is always present.
discovery
Status bar at the bottom of each product persistently shows the 2 most useful shortcuts (? and ⌘+K). New users learn shortcuts here without opening the sheet.
tooltips
Every clickable action with a shortcut shows it in its tooltip on hover. Dark indigo background, key tokens flip to dark variant. Label first, keys aligned right.
chord vs combo
Combo (held simultaneously): ⌘+S. Chord (sequential): G then D — separator is the word "then" in muted text, not a plus sign. Chord prefix shows a transient HUD ("G then…") at 300ms.
collisions
Never override platform shortcuts (⌘+W, ⌘+T). Don't bind Tab to a custom action — it always cycles focus.
platform
Show ⌘ on macOS, Ctrl on Windows / Linux. Detect with navigator.platform. Don't show both with a slash — pick one per session.
a11y
Cheatsheet is a focus-trapped modal. Esc closes. Filter input auto-focuses on open. Every shortcut is also available via menu / button — keyboard is an accelerator, never the only path.
