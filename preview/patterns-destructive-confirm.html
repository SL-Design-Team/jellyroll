headline
Always a question. Sentence case. Names the object type ("delete this pipeline?", "delete this environment?"). Never starts with "Confirm" or "Are you sure" alone — always specifies the action.
body
Names the specific object in quotes and bold ("'MG-pipeline-Nov2022'"). Describes the consequence (Recycle Bin? permanent? affected dependents?). Cites the retention window if soft-delete.
buttons
Soft delete: secondary "Cancel" + danger "Yes, delete". Save/discard: positive-phrased "No, don't save" + primary "Yes, save" — never "OK / Cancel". Destructive action always on the right.
color
Danger button is SL Red 600, filled, only for permanent or hard-impact actions. For soft-delete + 30-day undo, danger is still appropriate but the body text must mention the recovery window.
type-to-confirm
Required when action is permanent (env deletion, org delete, force-merge) or affects ≥5 users / ≥10 pipelines. User types the exact object name; confirm button stays disabled until match. Show a small impact summary (count of dependents) above the input.
focus
Modal opens with focus on the safest action (Cancel) — never the destructive button. Esc closes the modal as cancel (same behavior as the X in the corner). Enter activates Cancel by default.
after
Show a toast confirming the action ("Pipeline moved to Recycle Bin · Undo") with a 10s undo window when soft-delete. For permanent actions, no undo — show a confirmation toast only.
