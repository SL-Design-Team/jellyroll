frame
1440 × 900. Two-row grid: 56px navbar (full width) + flex body. The body is itself a two-column grid: rail (variable width) + right column. The 4px brand gradient sits under the navbar.
rail
Flush with the underside of the global navbar — runs full body height. 280px with search + filter segmented control + Snaps tree + My Assets tree + Open Project Manager footer. Collapses to a 48px icon column via the bottom chevron (not shown in these states).
right column
Owns toolbar, tabs+canvas, and status bar. None of these extend under the rail — they begin at the rail's right edge and run to the frame's right edge.
toolbar
Light surface, lives only inside the right column. Default (collapsed) state shows icons only at compact height. Expanding reveals group headers (Build / Execution / Configuration / Version Control / Transfer) and a label under each icon — toggled by the right-end chevron. Execute = blue, Validate = green; everything else is Indigo 900.
status bar
32px grey strip pinned to the bottom of the right column only — never under the rail. Container-query responsive: at ≤1080px right-column width, the hint cluster collapses to a single lightbulb icon (tooltip carries the full hint).
canvas
White, flex-centered hint copy when empty. Zoom controls dock bottom-left in a small elevated stack (fit / + / − / percent). Right-edge tab pulls in a right panel.

when to fork
This is the baseline empty state. Fork to mock specific scenarios: pipeline-with-snaps, validation-failed, run-in-progress, AI-assisted-build. Don't fork to redesign the chrome — that's a system change.
